# Terminal Setup Notes

This file is optional operator context, not an app build contract.

## Required For This Repo

- Bun `1.3.13+`
- Node.js `22+`
- Git
- Vercel CLI access for production deploys

The app does not require the personal shell aliases, Homebrew inventory, or machine snapshot that used to live in the root `zshrc.md`.

## Local Workflow

```bash
bun install
bun run dev
bun run check:full
```

## YEN Context

This portfolio is designed and tested primarily in YEN, but any modern terminal works. Keep YEN-specific setup in local machine docs unless it changes a repo workflow.
